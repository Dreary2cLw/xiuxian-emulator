## 安装      

> 在Yunzai-Bot/目录下执行  
```
git clone  https://gitee.com/hutao222/DDZS-XIUXIAN-V1.2.4/.git ./plugins/xiuxian-emulator-plugin/

```
> 如其他版本转1.2.4（注:1.2.4以上版本不兼容,数值完全不同）             
> 转移存档            
![转移存档](pic/photo4.jpg)            
这几个文件保存后将原版删除并执行安装命令随后将这几个文件替换到对应文件,然后执行【#一键同步】
